# agent-ia


# Clone o repositório:
```
git clone https://github.com/josiasdev/agent-ia
```

### Entre na pasta:
```
cd agent-ia/
```

### Instalação das dependências:
```
npm install
```

# Rodar o projeto:

## Entre na pasta:

```
cd agent-ia/
```

### Roda usando o npm-run-all:
```
npm run dev
```
